Gestor Web de Listas de Precios

Tecnologías Utilizadas

Este gestor de listas de precios es un proyecto FullStack desarrollado con las siguientes tecnologías:

  • MySQL: Base de datos relacional.
  • React: Biblioteca para construir interfaces de usuario.
  • Node.js: Entorno de ejecución para JavaScript del lado del servidor.
  • Express: Framework para construir aplicaciones web en Node.js.
  • JavaScript: Lenguaje de programación principal.
  • HTML y CSS: Para el diseño y estilo de la interfaz.

Características

En este proyecto el usuario tiene la capacidad de:

  • Crear Proveedores: Navega a la sección de proveedores y utiliza el formulario para agregar nuevos proveedores y asignarles una fórmula matemática.
  • Subir Productos: Usa la opción de carga de listas CSV para subir listas de productos. (Archivo delimitado por comas)
  • Filtrar y Buscar: Utiliza las opciones de filtrado y la barra de búsqueda en tiempo real para encontrar productos específicos en la lista.
  • Eliminar Productos: Selecciona productos de la lista y usa la opción de eliminación.

Validaciones y Casos de Uso

Este sistema gestor tiene la capacidad de detectar y validar los siguientes casos de uso al momento de subir productos:

  • Detectar cuando un producto ya está creado en la base de datos y descartarlo.
  • Detectar cuando el nuevo precio de un producto es mayor que el actual y actualizarlo.
  • Detectar cuando el nuevo precio de un producto está por debajo del actual y actualizarlo.
  • Detectar cuando un producto no necesita ser actualizado y descartarlo.
  • Detectar un nuevo producto y crearlo en la base de datos.
